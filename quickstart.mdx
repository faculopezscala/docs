## Overview

Scapester is an API service that transforms web content into clean, structured data formats. Simply provide a URL, and Scapester will return the content in your preferred format - whether that's markdown, structured JSON, or raw HTML.

## Installation

<CodeGroup>
  <CodeBlock label="npm" active>
    ```bash
    npm install scapester
    ```
  </CodeBlock>
  <CodeBlock label="pip">
    ```bash
    pip install scapester-py
    ```
  </CodeBlock>
</CodeGroup>

## Basic Usage

<CodeGroup>
  <CodeBlock label="Python" active>
    ```python
    from scapester import ScapesterApp

    app = ScapesterApp(api_key="sk-YOUR_API_KEY")

    # Scrape a website
    result = app.scrape_url(
        'https://example.com',
        params={'formats': ['markdown', 'html']}
    )
    print(result)
    ```
  </CodeBlock>
  <CodeBlock label="JavaScript">
    ```javascript
    import { ScapesterApp } from 'scapester';

    const app = new ScapesterApp('sk-YOUR_API_KEY');

    // Scrape a website
    const result = await app.scrapeUrl(
        'https://example.com',
        { formats: ['markdown', 'html'] }
    );
    console.log(result);
    ```
  </CodeBlock>
</CodeGroup>

## Response Format

```json
{
  "success": true,
  "data": {
    "markdown": "# Welcome to Example\nThis is the main content...",
    "html": "<!DOCTYPE html><html><body>...</body></html>",
    "metadata": {
      "title": "Example Website",
      "description": "An example website description",
      "language": "en",
      "sourceURL": "https://example.com",
      "statusCode": 200
    }
  }
}
```

## Advanced Features

### Page Interactions
Scapester allows you to interact with pages before scraping. This is useful for handling dynamic content or login flows:

<CodeGroup>
  <CodeBlock label="Python" active>
    ```python
    result = app.scrape_url('https://example.com', 
        params={
            'formats': ['markdown'],
            'actions': [
                {"type": "wait", "milliseconds": 1000},
                {"type": "click", "selector": "#login-button"},
                {"type": "wait", "milliseconds": 1000},
                {"type": "scrape"}
            ]
        }
    )
    ```
  </CodeBlock>
</CodeGroup>

### Structured Data Extraction
Extract specific data using custom schemas:

<CodeGroup>
  <CodeBlock label="Python" active>
    ```python
    from pydantic import BaseModel

    class ProductSchema(BaseModel):
        name: str
        price: float
        description: str

    result = app.scrape_url('https://example.com/product', {
        'formats': ['extract'],
        'extract': {
            'schema': ProductSchema.model_json_schema()
        }
    })
    ```
  </CodeBlock>
</CodeGroup>
